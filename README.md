# AutoEject Desktop Releases

This repository hosts the release binaries for AutoEject Desktop.

## Download

Download the latest version from the [Releases](https://github.com/balin77/autoeject-releases/releases) page.

## Features

- Process multi-plate 3D printing projects from Bambu Lab and Orca Slicer
- Automated plate swapping
- GCODE optimization
- Support for X1, P1, and A1M printers
- Automatic updates

## Installation

1. Download the latest `.exe` installer from Releases
2. Run the installer
3. The app will automatically check for updates on startup

## Source Code

The source code is maintained in a separate repository.
