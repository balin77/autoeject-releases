# AutoEject Desktop

Multi-Plate GCODE Post-Processor for Bambu Lab Printers.

## Download

Download the latest version from the [Releases](https://github.com/balin77/autoeject-releases/releases) page.

## Features

- Process multi-plate 3D printing projects from Bambu Lab and Orca Slicer
- Automated plate swapping
- GCODE optimization
- Support for X1, P1, and A1M printers
- Automatic updates

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```
